This matrix diagram visualizes common text passages in the CSEL corpus of the Churchfather writings.

Each colored cell represents a sequence alignment between two texts; darker cells indicate larger alignments.

Use the drop-down menu to reorder the matrix and explore the data.

Built with d3.js.

Source: Mike Bostock.

A network of text reuse can be represented by an adjacency matrix, where each cell ij represents an edge from vertex i to vertex j. Here, vertices represent bodies of text, while edges represent sequence alignments between the texts.
